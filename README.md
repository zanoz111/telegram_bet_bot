# Telegram Бот для Ставок

Бот для создания ставок между двумя людьми в Telegram.

## Функционал

- ✅ Создание матчей в формате "команда1 vs команда2"
- ✅ Установка коэффициентов для каждой стороны
- ✅ Выбор стороны для ставки другим пользователем
- ✅ Просмотр своих матчей и активных матчей

## Установка

1. Клонируйте репозиторий или скачайте файлы

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Создайте файл `.env` в корне проекта:
```
BOT_TOKEN=your_telegram_bot_token_here
```

4. Получите токен бота:
   - Напишите [@BotFather](https://t.me/BotFather) в Telegram
   - Используйте команду `/newbot` для создания нового бота
   - Скопируйте полученный токен в файл `.env`

## Запуск

```bash
python bot.py
```

## Использование

1. **Создание матча:**
   - Используйте команду `/create_match`
   - Введите матч в формате: `inz vs troolz`
   - Введите коэффициенты в формате: `1.5 2.3`

2. **Просмотр матчей:**
   - Используйте команду `/my_matches`
   - Вы увидите свои созданные матчи и активные матчи для ставок

3. **Выбор ставки:**
   - Через команду `/my_matches` выберите активный матч
   - Выберите сторону для ставки

## База данных

Бот использует SQLite для хранения данных. База данных создается автоматически при первом запуске.

Таблица `matches` содержит:
- Информацию о создателе матча
- Команды и коэффициенты
- Информацию о ставщике и выбранной стороне
- Статус матча
